# 朋友的妹妹只喜欢烦我

《朋友的妹妹只喜欢烦我》是一部校园恋爱喜剧，围绕大星明照、总爱对他撒娇拌嘴的小日向彩羽，以及他身边的创作伙伴展开。
作品一边描写吵吵闹闹的人际关系，一边把恋爱拉扯与游戏创作社团的日常交织在一起。
