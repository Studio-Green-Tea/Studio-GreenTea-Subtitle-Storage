# Chou Kaguya-hime

《Chou Kaguya-hime》是一部将《竹取物语》重新解构为近未来音乐奇幻的长篇动画电影。
故事围绕女高中生酒寄彩叶与来自月球的少女辉夜展开，把虚拟世界、直播竞赛与青春成长交织在一起。
