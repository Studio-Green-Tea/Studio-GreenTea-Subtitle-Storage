# Mayonaka Heart Tune

《Mayonaka Heart Tune》是一部围绕广播与青春梦想展开的动画。
故事讲述了主角遇到四位立志从事广播行业的少女，决心帮助她们实现梦想并寻找神秘人物阿波罗的真实身份。
