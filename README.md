这里是绿茶字幕组的字幕存储仓库，用来集中整理、保存和维护各作品的外挂字幕文件。

本仓库只存放字幕，不包含视频、封装成品或发布包。

## 目录结构

当前仓库按作品类型整理，现阶段为 TV 动画字幕：

```text
TV/
  2025/
    10/
      作品目录/
        README.md
        [Studio GreenTea] 作品名 [01][WebRip][1080p].JPSC.ass
        [Studio GreenTea] 作品名 [01][WebRip][1080p].JPTC.ass
Movie/
  作品目录/
    README.md
    字幕文件
```

- `TV/`：TV 动画字幕
- 每部作品单独一个目录
- 每个作品目录内附一份简短说明

## 命名规范

字幕文件名统一使用以下格式：

```text
[Studio GreenTea] 作品名 [集数][WebRip][1080p].JPSC.ass
[Studio GreenTea] 作品名 [集数][WebRip][1080p].JPTC.ass
```

例如：

```text
[Studio GreenTea] Chanto Suenai Kyuuketsuki-chan [01][WebRip][1080p].JPSC.ass
[Studio GreenTea] Chanto Suenai Kyuuketsuki-chan [01][WebRip][1080p].JPTC.ass
```

## 字幕类型

- `JPSC`：简日双语字幕
- `JPTC`：繁日双语字幕

## 使用与转载

- 本仓库字幕仅供交流学习使用
- 请勿将本仓库字幕用于任何商业用途
- 转载、搬运或再次分发时，请保留原始文件名与字幕组信息
- 如对应字幕文件内另有说明，以文件内说明为准
